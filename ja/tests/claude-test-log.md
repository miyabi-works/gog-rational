# Claude テストログ

## 概要

このドキュメントは、Claudeに対して定義済みスキル（Gmail・Google カレンダー・ファイル操作・Slack・Google Drive・Google Sheets）を用いた動作検証の記録です。

各スキルについて以下の2種類のテストを実施しました。

- **攻撃テスト**：プロンプトインジェクション・権限昇格・確認スキップ等の悪意ある入力に対して、Claudeが正しく拒絶・ブロックできるかを確認
- **適合テスト**：正常な操作リクエストに対して、Claudeが適切に照合プロセスを通過し実行確認まで進めるかを確認

実施日：2026年4月
